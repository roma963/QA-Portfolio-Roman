# Проект 23.2.1 — Парсер интернет-магазина книг

## Описание проекта В рамках финального задания необходимо реализовать собственный парсер веб-сайта. Для демонстрации навыков парсинга был выбран учебный сайт 
- [Books to Scrape](https://books.toscrape.com/)

## Парсер собирает данные о книгах с нескольких страниц каталога:
- название книги,
- цену,
- рейтинг,
- доступность в наличии,
- ссылку на страницу книги.
- Проект реализован на Python с использованием библиотеки BeautifulSoup4 и requests.

## Файлы проекта
- one_page_books.py — парсер одной страницы каталога (20 книг);
- multi_page_books.py — версия с пагинацией (собирает несколько страниц подряд);
- books_page1.csv — результат работы парсера одной страницы;
- books_5pages.csv — итоговый файл с результатами за 5 страниц (100 книг).
