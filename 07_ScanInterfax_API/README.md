# Проект: API тестирование (СКАН-Интерфакс)

Данный проект выполнен в рамках практического задания SkillFactory.  
Задача — протестировать часть API сервиса «СКАН» (Интерфакс) с использованием Postman, переменных коллекции, авторизации и данных из CSV.

---

##  Что реализовано

### 1. Авторизация пользователя
- Созданы переменные `test_login` и `test_password`.
- Через CSV выполняется параметризация позитивных и негативных тестов.
- Выполнено 3 позитивных и 5 негативных тестов логина.

### 2. Динамическая авторизация коллекции
- Все запросы автоматически подставляют токен из запроса «Авторизация пользователя».
- Токен сохранён в переменную `token`.

### 3. Получение текущего баланса
- Выполнен запрос `/balance`.
- Значение сохранено в переменную `current_balance`.

### 4. Получение тем публикаций
- Найдены три первых темы.
- Созданы переменные  
  `subjects_name1`, `subjects_name2`, `subjects_name3`  
- Количество тем сохранено в переменную `subjects_total`.

### 5. Получение документов по двум идентификаторам
- Переданы два ID (по заданию).
- Из ответа извлечены названия агентств.
- Созданы переменные `news_agent_1` и `news_agent_2`.
- Тест проверяет статус-код 200.
