#  Портфолио QA — Roman

## Добро пожаловать в моё портфолио проектов по тестированию!
Здесь собрано 18 практических работ, охватывающих все ключевые направления QA:

- Тестирование веб-приложений (UI/Web Testing)

- REST API тестирование

- Selenium автоматизация

- Мобильное тестирование

- Postman коллекции

- Автотесты на Python

- Парсинг и работа с JSON

- Тест-дизайн: чек-листы, тест-кейсы, баг-репорты

- Каждая папка — это законченный учебно-практический проект, оформленный с README, кодом и артефактами.

  ## Структура портфолио
01_Excursium_Autotests

Автоматизация UI-тестов веб-приложения Excursium (Selenium + Python).

02_PresiDent_TestDocumentation

Тестовая документация: чек-листы, тест-кейсы, баг-репорты.

Пользовательские истории и карта взаимодействий Yandex Taxi.

Кросс-браузерное тестирование сайта TASS.ru.

05_Postman_SkladTrade

06_PetFriends_API_Postman

Позитивные и негативные API-тесты PetFriends в Postman.

07_ScanInterfax_API

Тестирование публичного API Scan Interfax.

08_Leonardo_MobileTesting

Тестирование мобильного приложения "Леонардо".

09_PetFriends_ComplexTesting

Комплексная проверка API + UI PetFriends.

10_MultiplicationTable_NestedLoops

Учебный Python-скрипт с вложенными циклами (таблица умножения).

11_ProductOfOddNumbers_Loop

Python-скрипт: произведение нечётных чисел.

12_task18.7 — Консольный дневник

Консольное приложение на Python для работы с оценками.

13_tests_task_Detmir

Тест-кейс проекта «Детский мир».

14_json_orders_project

Проект работы с JSON: чтение, фильтрация, модификация данных.

15_Books_to_Scrape

Парсер интернет-магазина книг (Python + requests + bs4).

16_task24.7.2 — REST API

Негативные и граничные тесты REST API.

17_task25.4.1 — Пользовательские сценарии

Тест-кейсы бизнес-логики: цепочки вызовов API.

18_petfriends_Ожидания в Selenium_tests

Selenium-тесты: implicit и explicit waits в PetFriends.
