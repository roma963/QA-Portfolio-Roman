# Обработка данных заказов из интернет-магазина (JSON)
## Проект для практики работы с файлами формата JSON.
- Программа читает файл orders_july_2023.json и вычисляет:

1. Номер самого дорогого заказа
2. Заказ с наибольшим количеством товаров
3. День, когда было сделано больше всего заказов
4. Пользователя с наибольшим количеством заказов
5. Пользователя с наибольшей суммой заказов
6. Среднюю стоимость заказа
7. Среднюю стоимость товара
[Ссылка на проект](https://github.com/roma963/json_orders_project.git)
