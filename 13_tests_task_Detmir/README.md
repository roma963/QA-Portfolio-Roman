# Тестовое задание — Детский Мир
## Папка содержит:

- Чек-лист проверки карточки товара

- Баг-репорты (оформленные по приоритетам: Blocker / Critical / Major / Minor)

- REST-запросы из Postman (GET /products/{id}, POST /cart/items)

- SQL-запрос для сортировки заказов

- Скриншоты выполнения запросов

- Комментарии и выявленные несоответствия
- Мотивационное письмо
