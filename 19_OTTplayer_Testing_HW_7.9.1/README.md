## Краткое описание багов

### HT-13
- Тип: серверная ошибка
- Приоритет: высокий
- Серьёзность: критическая
- Результат: пользователь не может скачать приложение из-за ошибки 503

### HT-15
- Тип: ошибка ссылки
- Приоритет: средний
- Серьёзность: основная
- Результат: загрузка файла не начинается

### HT-16
- Тип: UI/UX дефект
- Приоритет: низкий
- Серьёзность: незначительная
- Результат: ухудшен пользовательский опыт
