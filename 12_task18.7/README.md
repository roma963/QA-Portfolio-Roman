# School Diary — консольный дневник ученика

- (Учебное задание 18.7)

Небольшой учебный проект на Python, реализующий простой консольный дневник, в котором можно управлять учениками, предметами и оценками.

## Проект демонстрирует работу с:

- циклами

- условиями

- структурами данных (словари, списки)

- пользовательским вводом

- модульной организацией кода

## Функциональность

- Программа позволяет:

Добавлять оценку ученику по предмету

Показывать оценки одного ученика

Добавлять нового ученика

Удалять ученика

Добавлять новый предмет

Удалять предмет

Выходить из программы

## Как работает программа

При запуске отображается меню команд.
Пользователь выбирает номер команды и вводит необходимые данные (имя ученика, название предмета, оценку и т.п.).

Все изменения хранятся в памяти программы

## Стек технологий

- Python 3.x

- Работа со словарями и списками

- Консольный ввод/вывод
