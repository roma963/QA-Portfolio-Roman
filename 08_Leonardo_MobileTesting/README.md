# Тестирование мобильного приложения «Леонардо»

Практическое задание по модулю «Основы тестирования мобильных приложений».  
Приложение выбрано по условию задания SkillFactory. Проверка выполнялась на Android-устройстве.

---

##  Цель работы

Провести исследовательское и функциональное тестирование мобильного приложения и составить следующие артефакты:

- Чек-лист по видам мобильного тестирования  
- Тестирование по каждому пункту  
- Баг-репорты на найденные дефекты  

---

##  Вид тестирования по заданию

1.  **Тестирование связи**  
2.  **Тестирование прерываний**  
3.  **Тестирование non-stop взаимодействий**  
4.  **Тестирование геолокации и GPS**  
5.  **Тестирование потребления ресурсов устройства**  
6.  **Инсталляционное тестирование**

##  Найденные дефекты

В ходе тестирования были выявлены различные проблемы в работоспособности приложения.  
Среди них:

- Авторизация и регистрация — невозможность входа / ошибка процесса;  
- Ошибка работы при отключении интернета;  
- Неправильная адаптация интерфейса при повороте экрана;  
- UX-проблема при оформлении заказа;  
- Некорректное определение местоположения;
- Ошибка отображения ближайших магазинов;
- Неверные варианты доставки по геолокации

Все баги оформлены по правилам: **шаги → фактический результат → ожидаемый → окружение → вложения**.
